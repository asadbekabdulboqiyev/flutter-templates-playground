Instructions — Flutter loyiha qanday boshlash kerak

Loyiha yaratish

Yangi Flutter loyiha yaratish uchun terminalni oching va quyidagi buyruqlarni kiriting:
flutter create my_app
cd my_app && flutter run

Paketlar o'rnatish

Kerakli paketlarni qo'shish:
flutter pub add provider
flutter pub add dio
flutter pub add flutter_riverpod
flutter pub add flutter_bloc

Struktura yaratish

Tizimli papka tuzilmasi:
mkdir lib/pages lib/services lib/widgets
mkdir lib/providers lib/models lib/utils

Build & Deploy

Release versiya yaratish:
flutter build apk --release
flutter build ios --release
flutter build web --release

Test & Format

Sifatni nazorat qilish:
flutter test
flutter analyze
dart format lib/

State Management

Tanlov: Provider, Riverpod, BLoC, GetX yoki Cubit. Har biri o'ziga xususiyatlarga ega. Boshlang'ich loyihalar uchun Provider, kattalar uchun Riverpod yoki BLoC tavsiya etiladi.

API integratsiya

Dio yoki http paketidan foydalaning. REST API, GraphQL, WebSocket qo'llab-quvvatlanadi. Firebase bilan ham ishlash mumkin.

UI & Animations

Material 3, Cupertino, yoki custom dizayn. Implicit animatsiyalar, Hero, CustomPainter, Lottie effektlari mavjud.
